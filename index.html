C# mtgdb.info

C# driver for api.mtgdb.info

Download .zip Download .tar.gz View on GitHub

  1. To install, Compile and create a reference to MtgDb.Info.Driver.dll or use NuGet: search for mtgdb.info

  2. The driver depends on Json.net, Via NuGet Package Manager Get Json.net, Dependencies are added if you used the mtgdb.info NuGet package.

  3. Smile and enjoy access to the mtg database :)
